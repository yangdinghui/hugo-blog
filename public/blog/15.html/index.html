git命令创建新分支
2024-11-15 分类： 技术 标签： #git
    • 1.切换到 master 分支‌（确保你当前在 master 分支上）
    • 2.从 master 分支创建一个新分支‌（假设新分支名为 new-branch）：
    • 3.将新分支推送到远程仓库‌
    • 4.完整的命令序列如下：

1.切换到 master 分支‌（确保你当前在 master 分支上）

git checkout master

2.从 master 分支创建一个新分支‌（假设新分支名为 new-branch）：

git checkout -b new-branch-name

3.将新分支推送到远程仓库‌

git push origin new-branch-name

4.完整的命令序列如下：

git checkout master
git checkout -b new-branch-name
git push origin new-branch-name
